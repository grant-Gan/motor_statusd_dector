# 电机健康状态检测系统

该项目为基于ESP32 Arduino框架开发, ESP32通过热电偶与振动传感器检测电机四个点的震动状态以及温度判数据; 而后将数据上传致One Net云平台, 进行数据可视化显示与报警阈值的动态设置.
